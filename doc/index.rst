欢迎使用Pygame Zero
======================


Pygame Zero可以让人们轻松无障碍创建游戏。

它是出于教育目的开发的，所以老师们可以用pygame zero讲授基本的编程知识，同时又不需
要解释复杂的Pygame的API和事件循环。

欢迎关注公众号，少儿创客帮，或者加微信 ilangxm 加入 pgzero 讨论群。
